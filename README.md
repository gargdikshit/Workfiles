Vagrant-3-tier-architecture
An all in one playbook to deploy a simple app and database using vagrant


Requirements
Vagrant 2.x.x


To run playbook
$ git clone https://github.com/gargdikshit/Workfiles.git
$ cd Wokfiles/Vagrant/3-tier
$ vagrant up 

OS Used:Ubuntu
Database used: Mongodb
Language/framework used: Node,Java, Maven

After vagrunt up:
3 Vm's will installes - Haproxy, app, db
Application will be cloned from git and deployed
